{{'Advanced' | translate}}
{{'Use Unconfirmed Funds' | translate}}
If enabled, wallets will also try to spend unconfirmed funds. This option may cause transaction delays.
{{'Use Polis Copay Style Addresses' | translate}}
If enabled, Polis addresses will be shown using Copay style address, and not the new cashaddr format.
{{'Use Dash Copay Style Addresses' | translate}}
If enabled, Dash addresses will be shown using Copay style address, and not the new cashaddr format.
{{'Recent Transaction Card' | translate}}
If enabled, the Recent Transactions card - a list of transactions occuring across all wallets - will appear in the Home tab.
